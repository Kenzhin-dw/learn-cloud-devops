- Belajar: whoami, pwd, ls, mkdir, echo
- Echo 
Echo is for displaying text it will say. Example Command Echo "Hello Kenzhin"
and it will say Hello Kenzhin
- Date
Date is for display date right now, Example Command is date
- Cal
cal is for display calender, Example Command is Cal/ Call 2025
- Expr 
Expr is for expression is for calculation for parameter, simple using is like expr 21 + 23, expr 2012 / 2, expr 123 % 3
- Figlet
figlet is for creating text we want, to convert in ASCII text, Command figlet "Kenzhin" or figlet -f slant "Kenzhin". -f parameter is for font style, and slant is a font that creates a slanted text effect
- clear
is for clearing the terminal we using
- ls
ls is for list of the current directory now, it shows you all the file and directory 
ls *.txt * means any file name ending with .txt
ls note* untuk * matches number characters
? single character
ls --help

- pwd
pwd is for print working directory ibarat kita lagi cek lagi di direktori mana gitu
- cd
cd adalah change directory, untuk mengganti atau masuk kedalam sebuah direktori ataupun folder , cd linux_practice
- mkdir
mkdir adalah make directory untuk membuat direktori/folder baru didalam linux sistem itu sendiri mkdir, linux_practice
- touch
touch adalah command untuk membuat sebuah file kosong tetapi kita tidak langsung mengeditnya, hanya membuat file kosong, touch hello.txt
touch note_{1..5}.txt untuk ngebuat file note dari 1-5

- command > 
adalah untuk mengirimkan sebuah kata kedalam suatu file, echo "Hello Linux" > hello.txt 
- cat
cat adalah concatenate untuk menampilkan isi dari file
- cp
cp adalah meng copy sebuah file
- rm
rm adalah file permission untuk menghapus sebuah file
- mv
mv adalah file permission untuk memindahkan sebuah file ke folder yang diinginkan
- tail -f /dev/null
command/perintah ini adalah untuk menjalankan sebuah proses yang berjalan terus menerus tanpa melakukan aktivitas apapun. -f adalah follow atau memaksa perintah tetap terbuka dan memantau secara real time. /dev/null ini seperti lubang hitam, file ini kosong.
- man
man adalah perintah manual berisi penjelasan command yang bisa dijalankan dengan perintah tersebut
